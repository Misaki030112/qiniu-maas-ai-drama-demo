# 项目路线图

这个仓库当前服务一个目标：尽快把“点众 AI 真人剧 Demo”做成一个可演示、可回看、可比较模型效果的最小闭环。

## 第一阶段

当前已完成：

- 引入 `Next.js` 项目制工作台。
- 左侧支持创建项目并进入项目工作台。
- 中间改成“故事输入 -> 分阶段执行”的流程，而不是一把执行。
- 已打通 `剧本改编` 阶段的真实执行。
- 已打通 `角色设定` 阶段的状态流和角色参考图生成链路。
- 支持手动修改 `adaptation / characters / storyboard`，并让下游阶段自动失效。

## 第二阶段

接下来优先推进：

- 完整验证 `分镜 -> 画面与配音 -> 输出合成` 的阶段链路。
- 补强项目页中对阶段结果的展示质量。
- 继续完善每个阶段的模型切换和结果对比。
- 把项目内的中间文件和状态说明整理得更清楚，方便演示。

## 第三阶段

后续增强项：

- 接入真正的视频模型阶段，而不是只保留占位。
- 强化角色一致性和镜头连续性。
- 增加多模型对比视图，帮助讲清楚模型取舍。
