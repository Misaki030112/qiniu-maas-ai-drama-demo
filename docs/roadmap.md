# Project Roadmap

这个仓库当前服务一个目标：尽快把“点众科技 AI 真人剧 demo”做成一个可演示、可回看、可比较模型效果的最小闭环。

## Phase 1

已完成：

- 输入剧本文本并输出短视频样片。
- 打通剧情改编、角色设定、分镜、画面生成、配音、字幕、合成。
- 保存每个节点的中间文件和模型信息。

## Phase 2

对应 Issue：

- [#1 Strengthen model comparison and role consistency](https://github.com/Misaki030112/qiniu-maas-ai-drama-demo/issues/1)

目标：

- 强化文本模型对比结果。
- 强化图像模型对比结果。
- 提升角色一致性和镜头连续性。
- 让运行目录能更清楚地解释每个节点为何选用某个模型。

## Phase 3

计划中的增强项：

- 用更适合真人剧的镜头素材生成方式替代纯静帧。
- 引入更细的角色语音策略，拉开不同角色的声音差异。
- 增加一页轻量展示页，直接回看每次运行的中间结果和最终样片。
